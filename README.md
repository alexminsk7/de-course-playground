# de-course-playground

A sandbox repository for learning Git and GitHub.

This is a test project created as the first assignment of a Design Engineering course.
Its only purpose is to practice the basic workflow: repository → commit → branch →
Pull Request → merge.

## What this repo is for

- Practicing everyday Git commands
- Working with branches and Pull Requests
- Getting familiar with the GitHub interface

No production code here — just experiments. 🧪
